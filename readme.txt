A plugin that will invalidate Cloudfront automatically on a post/page save.

== Installation ==
For Cloudfront Invalidation, you'll need to fill out 4 fields.

- Access Key ID
- Secret Key
- Cloudfront Distribution ID
- Region

== PHP Dependencies ==

PHP XML is required before using.  On a linux environment, it can be installed with the following command:
`sudo apt-get install php-xml`
